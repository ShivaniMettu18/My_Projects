Vision based rover integration

Integrating a vision-based rover for object manipulation and detection. Technologies like ROS have been used in this project. This project is helpful for specially-abled people to manage their daily works.

Weather App

Displaying the weather conditions, humidity in an area. This app uses frontend languages like html, css, javascript. It takes refernce from weather API and displays information. It also displays humidity, sunrise and sunset.

To-do List

The Simple To-Do List Web Application is a Node.js project designed to help users manage their tasks efficiently. The application allows users to create, read, update, and delete tasks from a list. The technology used is Node.JS framework basically written in javascript. It helps the users in organising the daily tasks.

Recipe Finder Application

The Recipe Finder Application is a basic user friendly project developed using Angular and React frameworks to help users discover, save, and share recipes from various sources. Users can search for recipes based on ingredients, cuisines, places, price or dietary preferences.
